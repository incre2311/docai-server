DOC·AI — Automated Documentary Production Tool
DOC·AI is an AI-powered documentary video production tool that automates the entire pipeline from script to rendered video.
What it does
	∙	Parses documentary scripts and breaks them into scenes
	∙	Classifies each scene as tense, emotional, or neutral
	∙	Auto-generates search queries to fetch relevant stock footage from Pexels/Pixabay
	∙	Renders final video with Ken Burns effects, color grading presets, and smart cutting rules
	∙	Deployed live on Railway using Flask + FFmpeg
Tech Stack
	∙	Backend: Python, Flask, FFmpeg
	∙	Deployment: Railway (Docker)
	∙	Frontend: HTML/CSS (tool.html)
	∙	AI Integration: Scene classification, auto query generation
Status
🚧 Active development — core pipeline functional, additional features in progress.
Use Case
Built for creators who want to produce cinematic documentary-style videos using AI assistance — from script input to final rendered output.
